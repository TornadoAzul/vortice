# Vórtice

Sistema de gestión personal y profesional basado en Obsidian, organizado según la metodología PARA (Projects, Areas, Resources, Archive).

## 📁 Estructura

```
vortice/
├── projects/       # Proyectos activos con fecha límite
├── areas/          # Áreas de responsabilidad continua
├── resources/      # Referencias y material de apoyo
├── archive/        # Proyectos y áreas completados/inactivos
├── clients/        # CRM - Gestión de clientes
├── inbox/          # Captura rápida de ideas y tareas
├── journal/        # Notas diarias y reflexiones
├── start/          # Dashboards y guías de inicio
└── templates/      # Plantillas para nuevos archivos
```

## 🛠️ Tecnologías

- **Gestión de Conocimiento**: Obsidian
- **Queries Dinámicos**: Dataview plugin
- **Versionamiento**: Git + GitHub
- **Sincronización**: Google Drive

## 📝 Uso

1. Los archivos se organizan automáticamente con frontmatter YAML
2. Dataview genera vistas dinámicas de proyectos, clientes y tareas
3. Los cambios se sincronizan automáticamente a GitHub
4. Las plantillas facilitan la creación consistente de nuevos archivos
